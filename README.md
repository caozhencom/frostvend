## 如何替换素材

1. 将真实图片放到 `website/images/` 文件夹
2. 将真实视频放到 `website/videos/` 文件夹
3. 在 `index.html` 中找到对应的 `image-placeholder` 块
4. 替换为：
   ```html
   <img src="images/your-image.jpg" alt="描述" class="real-image">
   ```
   或视频：
   ```html
   <video src="videos/your-video.mp4" controls></video>
   ```
